# Legacy Directory

This directory contains reference copies of all components, pages, and styles from the Next.js implementation of policestationagent.com. These files serve as a legacy reference for the complete site structure.

## Component Files

- `layout.tsx` - Root layout component with metadata
- `navbar.tsx` - Header/Navigation component (desktop & mobile)
- `footer.tsx` - Footer component with links and contact info
- `global.css` - Global styles and Tailwind configuration

## Page Components

- `home.tsx` - Homepage with hero, services, and police stations sections
- `blogList.tsx` - Blog listing page
- `blogPost.tsx` - Individual blog post page with JSON-LD schema
- `stationPage.tsx` - Individual police station page with LocalBusiness schema
- `servicesPage.tsx` - Services listing page
- `faq.tsx` - FAQ page with accordion-style questions
- `contact.tsx` - Contact page with emergency service information
